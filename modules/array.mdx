# Módulo `array`

Utilidades para manipular arreglos. Dado que Kadoku maneja variables inmutables por defecto, funciones como `push` devuelven un nuevo arreglo modificado.

### `array.len(arr)`
Devuelve la longitud del arreglo.

### `array.push(arr, value)`
Devuelve un nuevo arreglo con el `value` añadido al final.

### `array.pop(arr)`
Devuelve un nuevo arreglo sin su último elemento.

### `array.join(arr, sep)`
Une los elementos del arreglo en una sola cadena, usando `sep` como separador.

# `array` Module

Utilities for manipulating arrays. Since Kadoku handles immutable variables by default, functions like `push` return a new modified array.

### `array.len(arr)`
Returns the length of the array.

### `array.push(arr, value)`
Returns a new array with `value` appended to the end.

### `array.pop(arr)`
Returns a new array without its last element.

### `array.join(arr, sep)`
Joins the array elements into a single string, using `sep` as separator.
