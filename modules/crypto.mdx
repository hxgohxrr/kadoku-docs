# Módulo `crypto`

Funciones de criptografía y codificación básicas.

### `crypto.base64_encode(text)`
Codifica una cadena de texto en formato Base64.

### `crypto.base64_decode(encoded_text)`
Decodifica una cadena en formato Base64. (Actualmente soportado de forma experimental en el runtime minimalista).

# `crypto` Module

Basic cryptography and encoding functions.

### `crypto.base64_encode(text)`
Encodes a string into Base64 format.

### `crypto.base64_decode(encoded_text)`
Decodes a Base64 format string. (Currently supported experimentally in the minimal runtime).
