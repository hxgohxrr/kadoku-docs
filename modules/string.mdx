# Módulo `string`

Utilidades para manipular cadenas de texto.

### `string.len(s)`
Devuelve la longitud de la cadena.

### `string.split(s, sep)`
Divide la cadena utilizando el separador dado y devuelve un arreglo de cadenas.

### `string.replace(s, from, to)`
Reemplaza todas las ocurrencias de `from` por `to` dentro de la cadena.

### `string.to_lower(s)`
Convierte la cadena a minúsculas.

### `string.to_upper(s)`
Convierte la cadena a mayúsculas.

### `string.trim(s)`
Elimina los espacios en blanco al inicio y al final de la cadena.

# `string` Module

Utilities for manipulating strings.

### `string.len(s)`
Returns the length of the string.

### `string.split(s, sep)`
Splits the string using the given separator and returns an array of strings.

### `string.replace(s, from, to)`
Replaces all occurrences of `from` with `to` in the string.

### `string.to_lower(s)`
Converts the string to lowercase.

### `string.to_upper(s)`
Converts the string to uppercase.

### `string.trim(s)`
Removes whitespace from both ends of a string.
