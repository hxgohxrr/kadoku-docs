# Módulo `path`

Utilidades para manipular y analizar rutas de archivos de manera multiplataforma.

### `path.join(p1, p2)`
Une dos segmentos de ruta utilizando el separador del sistema operativo actual.

### `path.basename(p)`
Devuelve el nombre del archivo (incluyendo la extensión) al final de la ruta.

### `path.extname(p)`
Devuelve la extensión del archivo, por ejemplo `txt` o `ku`.

### `path.dirname(p)`
Devuelve el nombre del directorio que contiene el archivo especificado.
</Tab><Tab title="English">
# `path` Module

Utilities for manipulating and parsing file paths cross-platform.

### `path.join(p1, p2)`
Joins two path segments using the current operating system's separator.

### `path.basename(p)`
Returns the file name (including extension) at the end of the path.

### `path.extname(p)`
Returns the file extension, e.g., `txt` or `ku`.

### `path.dirname(p)`
Returns the directory name that contains the specified file.
