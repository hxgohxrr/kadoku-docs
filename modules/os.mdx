# Módulo `os`

Interactúa con el sistema operativo.

### `os.exit(code)`
Termina la ejecución del programa con un código de salida.

### `os.sleep(ms)`
Pausa la ejecución durante el tiempo especificado en milisegundos.

### `os.cwd()`
Devuelve el directorio de trabajo actual.

### `os.chdir(path)`
Cambia el directorio de trabajo actual. Devuelve `true` si tiene éxito.

# `os` Module

Interact with the operating system.

### `os.exit(code)`
Terminates program execution with an exit code.

### `os.sleep(ms)`
Pauses execution for the specified time in milliseconds.

### `os.cwd()`
Returns the current working directory.

### `os.chdir(path)`
Changes the current working directory. Returns `true` if successful.
