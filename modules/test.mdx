# Módulo `test`

Herramientas para escribir y ejecutar aserciones o pruebas unitarias en Kadoku.

### `test.assert_eq(a, b)`
Afirma que `a` y `b` son iguales. Si lo son, devuelve `true` y pasa la prueba; si no, imprime un error y devuelve `false`.

### `test.assert_true(a)`
Afirma que `a` es verdadero (`truthy`). Devuelve `true` si pasa; de lo contrario devuelve `false`.

# `test` Module

Tools for writing and running assertions or unit tests in Kadoku.

### `test.assert_eq(a, b)`
Asserts that `a` and `b` are equal. If they are, it returns `true` and passes; otherwise, it prints an error and returns `false`.

### `test.assert_true(a)`
Asserts that `a` is truthy. Returns `true` if it passes, `false` otherwise.
