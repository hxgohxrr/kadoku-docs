# Dependencias de Crates

Puedes incluir dependencias de `crates.io` en tu `project.ku.json` bajo el objeto `compiler.rust_dependencies`. Kadoku las inyectará automáticamente en el entorno de compilación de Rust.
  </Tab>
  <Tab title="English">
# Crate Dependencies

You can include dependencies from `crates.io` in your `project.ku.json` under the `compiler.rust_dependencies` object. Kadoku will automatically inject them into the Rust build environment.
