# Tipos de Datos

Kadoku maneja los siguientes tipos internamente:
- **Primitivos**: Numeros (enteros y flotantes de 64-bit), Strings, Booleanos (`true`, `false`).
- **Complejos**: Arreglos (`[1, 2, 3]`), Objetos/Diccionarios (`{a: 1}`), Tuplas (`(1, 2)`).
- **Nulos**: `null`, `undefined`.</Tab><Tab title="English">
# Data Types

Kadoku handles the following types internally:
- **Primitives**: Numbers (64-bit integers and floats), Strings, Booleans (`true`, `false`).
- **Complex**: Arrays (`[1, 2, 3]`), Objects/Dictionaries (`{a: 1}`), Tuples (`(1, 2)`).
- **Nulls**: `null`, `undefined`.</Tab>
