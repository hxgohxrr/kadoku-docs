# Inicio Rápido

Para empezar rápidamente con Kadoku:
1. Ejecuta `kadoku create mi_app`
2. Entra al directorio `cd mi_app`
3. Ejecuta tu código con `kadoku run`

¡Eso es todo! El compilador se encarga de descargar dependencias y generar el binario.</Tab><Tab title="English">
# Quickstart

To get started quickly with Kadoku:
1. Run `kadoku create my_app`
2. Enter the directory `cd my_app`
3. Run your code with `kadoku run`

That's it! The compiler takes care of downloading dependencies and generating the binary.</Tab>
