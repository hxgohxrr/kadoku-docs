    # ¿Qué es Kadoku?

    Kadoku es un lenguaje de programación funcional y de tipado dinámico, diseñado para ofrecer una experiencia de desarrollo fluida y productiva. Combina una sintaxis limpia inspirada en Python con una filosofía de ecosistema integrado al estilo Node.js.

    ## Características Principales

    - **Sintaxis Limpia:** Elimina el ruido visual. Usa indentación para bloques lógicos en lugar de llaves `{}`, haciendo el código más legible y elegante.
    - **Rendimiento Nativo:** Kadoku compila directamente a binarios nativos de Rust (`.exe` en Windows), ofreciendo un rendimiento excepcional.
    - **Ecosistema Integrado:** El compilador incluye todo lo necesario. No requieres herramientas de terceros para formatear, revisar errores (linting) o empaquetar código.
    - **Tipado Dinámico:** Las variables son dinámicas, pero el motor interno maneja los tipos de forma eficiente.
    - **Librería Estándar Robusta:** Módulos nativos súper rápidos (escritos en Rust) para sistema de archivos, red, base de datos e integraciones con IA.

    ## Filosofía

    Kadoku nace con la idea de proporcionar a los desarrolladores una herramienta que les permita centrarse en la lógica de su aplicación, sin preocuparse por la configuración del entorno, la gestión compleja de dependencias o el rendimiento final.     # What is Kadoku?

    Kadoku is a dynamically typed, functional programming language designed to offer a smooth and productive development experience. It combines clean syntax inspired by Python with an integrated ecosystem philosophy similar to Node.js.

    ## Key Features

    - **Clean Syntax:** Eliminates visual noise. It uses indentation for logical blocks instead of `{}` braces, making the code more readable and elegant.
    - **Native Performance:** Kadoku compiles directly to native Rust binaries (`.exe` on Windows), offering exceptional performance.
    - **Integrated Ecosystem:** The compiler includes everything you need. You don't need third-party tools to format, lint, or package code.
    - **Dynamic Typing:** Variables are dynamic, but the internal engine handles types efficiently.
    - **Robust Standard Library:** Super-fast native modules (written in Rust) for file system, networking, databases, and AI integrations.

    ## Philosophy

    Kadoku was born with the idea of providing developers with a tool that allows them to focus on their application's logic, without worrying about environment configuration, complex dependency management, or final performance. By compiling to Rust, you get the security and speed of a low-level language with the ease of use of a high-level script.</Tab>
