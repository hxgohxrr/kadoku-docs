    # La Interfaz de Línea de Comandos (CLI)

    El compilador de Kadoku actúa como una herramienta "todo en uno".

    ## Comandos Principales

    - **`kadoku create <nombre>`**: Crea un nuevo proyecto desde cero.
    - **`kadoku build [archivo] [--release]`**: Compila el código fuente `.ku` a un binario nativo de Rust.
    - **`kadoku run [archivo]`**: Compila y ejecuta tu programa en un solo paso.
    - **`kadoku fmt`**: Formatea automáticamente todos los archivos `.ku`.
    - **`kadoku lint`**: Ejecuta el linter estático integrado para revisar errores y antipatrones.

    ## Otros Comandos

    - `kadoku package`: Empaqueta tu librería.
    - `kadoku add <pkg>`: Instala un paquete del registro.

    # The Command Line Interface (CLI)

    The Kadoku compiler acts as an "all-in-one" tool.

    ## Main Commands

    - **`kadoku create <name>`**: Creates a new project from scratch.
    - **`kadoku build [file] [--release]`**: Compiles the `.ku` source code to a native Rust binary.
    - **`kadoku run [file]`**: Compiles and runs your program in a single step.
    - **`kadoku fmt`**: Automatically formats all `.ku` files.
    - **`kadoku lint`**: Runs the built-in static linter to check for errors and anti-patterns.

    ## Other Commands

    - `kadoku package`: Packages your library.
    - `kadoku add <pkg>`: Installs a package from the registry.
    - `kadoku test`: Runs all defined tests.</Tab>
